# NotificationInator
A tool to write and test Tornium notifications.

![My blow-itself-up-inator](https://media.tenor.com/85103ddR6rwAAAAM/phineas-and-ferb-heinz-doofenschmirtz.gif)

Credit for the project name goes to [DarthRevan](https://www.torn.com/profiles.php?XID=2836054).

## Installation
The package can be installed from GitHub by adding `notification_inator` to your list
of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:notification_inator, git: "Tornium/notification_inator"}
  ]
end
```
